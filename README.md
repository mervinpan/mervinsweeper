### Installation

1. Clone the repo
```sh
git clone https://github.com/mervinpan/mervinsweeper.git
```
2. Install NPM packages
```sh
npm install
```
3. Run compiler
```sh
npm run build
```
4. Start server
```sh
npm start
```
